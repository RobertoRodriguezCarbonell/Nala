# Landing de Nala

Landing page de producto (Next.js + Tailwind). Single-page, estática.

## Desarrollo

```
npm install
npm run dev
```

## Despliegue (Vercel)

Importa el repo en Vercel y pon **Root Directory = `landing/`**. Framework: Next.js
(autodetectado). No requiere variables de entorno.

El botón de descarga apunta a la release de GitHub (`/releases/latest`); publica una
release con el instalador para que funcione.
